## üö® Emergency Procedures

### If App Crashes
1. Check Railway logs
2. Look for error message
3. Ask AI in chat (if accessible): "Debug the error: [paste error]"
4. Redeploy from Railway dashboard
5. Check environment variables still set

### If Running Out of Credits
1. Check OpenRouter usage
2. Top up credits at openrouter.ai
3. Reduce workflow frequency temporarily
4. Monitor costs going forward

### If Videos Not Posting
1. Check Blotato dashboard
2. Verify account connections
3. Check account IDs in workflow config
4. Re-authenticate accounts if needed
